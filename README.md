# talkSPORT Companion

A Flutter app for listening to talkSPORT and talkSPORT 2 live streams and recent catch-up shows on Windows and Android.

## Release Builds

Build installable Windows and Android release assets with:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\package_release.ps1
```

Artifacts are written to `dist\release\vX.Y.Z`.

See `docs\INSTALL.md` for install and GitHub Releases instructions.
